## Steps

- give name to your project

- cd to your project
- npm install
- npm run dev
- follow link for run[http://localhost:5173/]

- src -- all source code or main code

    - index.jsx -- index file
    - vite.config.js
    - assets -- all images that are commonly used

- public -- all static files
    - all html files 
    - css files
    - images

- package.json

- vite.config.js

- index.html
